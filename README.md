# Machine Learning–Based Classification of Benign and Malignant Tumors

## 📌 Overview
This project compares several machine learning models to classify tumors as benign or malignant using the Breast Cancer Wisconsin Diagnostic Dataset.

---

## ⚙️ Methods
- Linear Discriminant Analysis (LDA)
- Quadratic Discriminant Analysis (QDA)
- Naive Bayes
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Random Forest
- Gradient Boosting

---

## 🔬 Feature Engineering
- PCA-based feature screening  
- Top 10, 15, 20, and 25 features selected using mean absolute loadings  

---

## ⚖️ Imbalance Handling
- Under-sampling  
- K-medoids clustering  
- SMOTE (Synthetic Minority Over-sampling Technique)  

---

## 📊 Evaluation
- Cross-validation  
- Accuracy  
- Area Under the Curve (AUC)  

---

## 📈 Key Results
- KNN achieved near-perfect classification performance  
- SVM, QDA, and Random Forest achieved approximately **98% accuracy**  
- Feature selection and imbalance handling significantly improved model performance  

---

## 🛠️ Tools & Technologies
- R (caret, e1071, randomForest, MASS, DMwR)  
- Machine learning and statistical classification techniques
